Trusted Compute
===================
Trusted Compute is a set of Software Defined Security extensions that utilize Hardware security
capabilities on the Edge Manageability Framework.

- Trusted Compute solution offers protection of the Edge Node, Workloads and Data at rest, in-transit and in-use on the Edge node.
- Continuous monitoring of the Edge Node using TPM Attestation.

Trusted Compute is a key aspect of |software_prod_name| that provides workload protections of the system. This section provides a
comprehensive view of |software_prod_name| Trusted Compute extension and an
overview of key components that help you monitor the trust status of the Edge nodes and protect the workloads by isolated execution.
